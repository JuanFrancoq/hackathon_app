# Project_inges
Proyecto final de Programación 3 para la HACKATHON elaborado por Juan José Franco, Miguel angel Londoño, Luisa Fernanda Cano.

El objetivo principal es desarrollar una aplicación distribuida en Elixir que facilite la organización y colaboración en la Hackathon. El sistema debe permitir la creación y gestión de equipos, el registro y visualización de proyectos, la comunicación en tiempo real ent
